Opening Doors is my Final Project for Code Louisville Data Analytics 2.

This project will allow the user to search a home value, and analyze rental and/or selling options based on the record of Realty Mole Property.
GitHub Repository

Requirements

Create an API key for free at Realty Mole Property

FEATURE LIST:

Category 1 - Loading data:
- Read TWO data sets in with an API (or use two different APIs that have data you can combine to answer new questions).

Category 2 - Clean and operate on the data while combining them:
- Cleaning data and performing a pandas merge with your two data sets, then calculate some new values based on the new data set.

Category 3 - Visualize / Present data:
- Make 3 matplotlib or seaborn (or another plotting library) visualizations to display your data.
- Make at least 1 Pandas

Category 4: Best Practices
- The program utilize a virtual environment and document library dependencies in a requirements.txt file.

“STRETCH” Feature List:
- Annotate your .py files with well-written comments and a clear README.md.
